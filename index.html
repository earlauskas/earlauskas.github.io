Projects

These are my most recent projects. They are just that-projects that are continuously evolving. Check them out!

PacMan Activity

PacMan Description: This activity uses a loop to create PacMen moving at random velocities on command. As the "Add a PacMan" button is clicked, a new PacMan is created, which moves in a random direction and speed. When a PacMan hits the edge of the screen, it bounces off of the edge and moves in the opposite direction. Clicking the "Start Game" button causes the PacMen to move around. Future Improvements: I would like to continue working on this project so that when the PacMan hits the edge of the screen, the image changes to face the opposite direction along with the movement.

Eye Movement Activity

Eye Movement Activity Description This activity is an exploration into mouse events, specifically with mousemove. My task was to add a second "eye" element that follows the mouse as it moves around the screen. Future Improvements: I would like to include more mouse events in this activity, namely having the background change color on mouseclick.

Bus Marker Activity

Bus Activity Description This activity focuses on pulling in a map of Cambridge from MapBox and adding custom markers to it along the #1 bus route between Harvard and MIT. This was my first time working with an access Token and integrating the MapBox JS library. Future Improvements: I would like to continute to add bus stops along the entire #1 bus route for the city of Boston. Then I would like to continue working to create stops along my commute to work.
